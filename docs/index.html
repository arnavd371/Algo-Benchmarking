Algo-Benchmarking

An interactive tool for exploring non-convex optimisation landscapes and benchmarking gradient-based algorithms. Select any of eight classic benchmark functions, visualise the 2-D landscape as a heatmap, then watch your chosen optimisers navigate the terrain in real time.

// 8 benchmark functions
Rastrigin · Ackley · Rosenbrock
Himmelblau · Levy · Schwefel
Griewank · Sphere

// 4 optimisation algorithms
Gradient Descent (Armijo LS)
SGD + Nesterov Momentum
Adam · RMSprop

// fully client-side — no server needed

What can you do?

Landscape Explorer

Visualise any of the 8 benchmark functions as an interactive 2-D contour heatmap. Toggle log scale, adjust resolution, and mark the global minimum.

Optimisation

Set a starting point (or click on the map), pick one or more optimisers, and watch their trajectories unfold over the landscape alongside live convergence curves.

Benchmark Compare

Run all selected optimisers from the same start and compare final objective values, iteration counts, and convergence behaviour with bar charts and tables.

Hessian Spectrum

Sample Hessian eigenvalues across the landscape to measure curvature. Discover how many directions are convex vs concave, and estimate saddle-point density.

Tip: The eigenvalues of the Hessian at a point tell you about local curvature. All-positive = local minimum; mixed signs = saddle point.
Eigenvalue statistics
Interpretation guide:
■ Positive eigenvalues indicate locally convex (bowl-shaped) directions.
■ Negative eigenvalues indicate concave directions — the point is a saddle point or local maximum.
■ Near-zero eigenvalues indicate flat regions where gradient-based methods slow down significantly.
